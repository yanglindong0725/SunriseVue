# D.Henry Vue3 项目模板

## 推荐的 IDE 设置

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur) 

## 前置条件

### Node.js

`node` 需要 `^16 || ^18` 版本

```sh
# 查看 node 版本
node -v
```

### [PNPM](https://pnpm.io/zh)

如果你没有安装过 `pnpm`

```shell
npm install pnpm -g
```

## 项目运行

```sh
# 安装依赖
pnpm install

# 开发环境运行
pnpm dev

# 生产环境构建
pnpm build

# 代码检查
pnpm lint
```
